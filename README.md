# Simple-Chat v0.6

### For simple chatting!

This project was built using:

- Material UI
- React
- Firebase (Realtime Database, Firestore and Auth)

And it's written in Typescript! ❤️

Users are able to:

- Register/Sign In
- Create/Join servers
- Create/Delete channels
- Use custom emotes
- Send images
- View who's online
- Chat!!!
